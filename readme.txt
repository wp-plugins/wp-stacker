== Description ==

**WP Pocket** is a Wordpress plugin, with which you can post interesting links from Pocket (Pocket.com) automatically on your blog - with autogenerated titles, discriptions and tags. You can customize the layout of the post completely. So you can save a lot of time, but show all your readers all your interesting links.

== Installation ==

1. Install automatically by clicking on the 'Plugins', 'Add New' menu in WordPress, or upload the 'wp-pocket' folder into the '/wp-content/plugins/' directory.
2. Activate the plugin with the 'Plugins' menu in WordPress
3. Configure the plugin by clicking on 'Options', 'WP Pocket'. Make sure the output is congruous with the laws in your country.

== Frequently asked questions ==

= How can I add my Pocket account? =

Click on 'Options', 'WP Pocket' in Wordpress and add your login datas.

= Can I use my own API-Key? =

Yes, you can, but you do not have to. To add one, click on 'Options', 'WP Pocket' in Wordpress and add the API-Key.

= Can I customize the gennerated post? =

Yes, you can customize it completely! The first way of customizing is to change the settings in 'Options', 'WP Pocket'. Later, when you generate a post, by clicking on 'WP Pocket' in the main menu, you can change the order, title, description, tags and the url of the links. The last option to customize it, is to edit the articel with the normal WordPress articel editor.
